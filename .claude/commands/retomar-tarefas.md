# Retomar Tarefas Pendentes

O usuário está iniciando uma nova sessão e quer retomar o trabalho anterior.

1. **Leia o arquivo** `.claude/pending-tasks.md` para entender o contexto.

2. **Verifique o estado atual** do repositório:
   - Execute `git status` para ver arquivos modificados
   - Compare com o que foi salvo no arquivo de tarefas

3. **Apresente um resumo** ao usuário:
   - O que foi feito na sessão anterior
   - O que ainda está pendente
   - Se há conflitos ou mudanças inesperadas

4. **Pergunte** ao usuário se deseja:
   - Continuar de onde parou
   - Começar uma tarefa diferente
   - Ver mais detalhes sobre algum item pendente

5. **Se continuar**, carregue as tarefas pendentes no TodoWrite para tracking.

6. **Após retomar**, sugira arquivar ou limpar o arquivo `.claude/pending-tasks.md`.
