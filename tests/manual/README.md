# Testes Manuais

Scripts para testes manuais de funcionalidades específicas.

**⚠️ NOTA:** Estes testes devem ser eventualmente convertidos em testes automatizados.

## Arquivos:
- **testar-api-tesouraria.js**: Teste manual da API de tesouraria
- **testar-top10-historico.js**: Teste do módulo Top10 histórico
- **test-quick-bar.js**: Teste da barra de acesso rápido

## Executar:
```bash
node tests/manual/testar-api-tesouraria.js
```
