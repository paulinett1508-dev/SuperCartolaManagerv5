# Utilitários de Documentação

Scripts para gerar documentação automática do projeto.

- **handover.js**: Gerador de handover (transição de projeto)
- **project-dna.js**: Mapeador inteligente completo do projeto
- **system-mapper.js**: Mapeador de sistema
- **ux-analyzer.js**: Analisador de padrões UX

## Uso:
```bash
node project-dna.js    # Gera PROJECT_DNA.md
node ux-analyzer.js    # Gera UX_PATTERNS.md
```
